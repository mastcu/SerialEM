DECTRIS Advanced Settings Dialog

This dialog allows some advanced setting to be selected for a DECTRIS camera.

Incident energy

In the combo box, you can select from among the available incident energy values for the particular camera, or select 'Auto' for an automatic selection.

Threshold energy

In the combo box, you can select from among the available threshold energy values for the particular camera, or select 'Auto' for an automatic selection.
