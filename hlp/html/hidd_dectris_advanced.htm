DECTRIS Advanced Settings Dialog

This dialog allows some advanced setting to be selected for a DECTRIS camera.

Incident energy

In the combo box, you can select from among the available incident energy values for the particular camera, or select 'Auto' for an automatic selection based on the last known microscope voltage.

Threshold energy

In the combo box, you can select from among the available threshold energy values for the particular camera, or select 'Auto' for an automatic selection.

Autosummation

Check this option to automatically subdivide long exposures, extending the camera's dynamic range.

Countrate correction

Check this option to apply built-in linearity correction.

Retrigger

Check this option to have non-paralyzable counting.

DCU internal flatfield correction

Check this option to use the DCU internal flatfield correction mechanism.

Bad pixel interpolation

Check this option to interpolate values for bad pixels using surrounding pixels.

Module gap interpolation

Check this option (available only for SINGLA cameras) to interpolate the gap between sensor modules.
