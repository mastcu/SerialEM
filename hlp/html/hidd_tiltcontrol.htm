Tilt Control Panel

This window displays the current tilt angle, allows one to tilt by an increment or to go to a desired angle, and contains options for tilt-related parameters.

Tilt Angle Readout

This readout shows the current tilt angle, which should match the tilt readout in the microscope user interface.

Tilt Up

Tilt Down

These buttons will tilt toward positive or negative tilt angles by the current tilt increment, which is either the basic tilt increment, or this basic increment multiplied by the cosine of the tilt angle if 'Cosine increments' is selected.

Set Increment

Use this button to set the basic tilt increment. A dialog box will appear, with the current value as the default. Enter a value in degrees.

Set Delay

Use this button to set the delay to be applied after tilting by the basic increment. A dialog box will appear, with the current value as the default. Enter a value in seconds. This delay time must expire before a new image is taken with the camera.  This applies when not doing a tilt series, or when doing a lower-magnification task during a tilt series (specifically, finding eucentricity); a separate delay set in the Tilt Series Setup dialog governs Record, focusing, and other images taken during a tilt series at the working magnification.

Cosine Increments

Use this button to select cosine tilt increments, in which the increment is proportional to the cosine of the tilt angle.
