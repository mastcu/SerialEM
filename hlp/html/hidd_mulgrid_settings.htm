Multiple Grid Settings Manager Dialog

This dialog lets you store separate settings for individual grids in five different categories.  The Final Data parameters in the Navigator Acquire at Items  dialog can be modified and stored in their entirety.  The dialog simply indicates whether these are stored or not for the current grid.  For autocontouring, hole finding, and multiple Record acquisition, a subset of parameters are stored.  The final category consists of a few other general program settings that would typically differ between gold and carbon grids.  Each section for the lower four categories has two summary lines: One labeled 'Current' showing the current settings, and one labeled 'Grid' showing the stored settings for the current grid in the Multiple Grid Operations dialog, if any, or simply saying that there are none.  The selected grid can be changed in that dialog and this dialog will respond appropriately.

Unlike in the Navigator dialog, there is no inheritance of modified parameters from one item to the next; separate settings must be stored for each grid needing them.

Final Data Acquire Settings

To set these parameters for one grid, simply press Set/View for Grid to open the Navigator Acquire at Items dialog, adjust the parameter settings there, and press the 'Close' button.  The settings will be stored without affecting the global settings to be used for the other grids.  The same button is used if you want to check or modify previously stored parameters.  The Revert to Global button can be used to discard the stored settings and retrun to using the globally set one.

To apply the same settings to other grids without having to make the individual changes over and over, press Apply Grid as Current.  Then select each other grid in turn, pressing Set/View for Grid and just closing the Navigator Acquire at Items dialog, which has the modified settings already.  When done, press Restore Current to restore the global parameters.

Other Parameter Categories

Setting parameters for the other categories is bit more cumbersome because this dialog does not have control over the other dialogs involved in those settings.  Here, the procedure to set parameters for a grid when none are stored yet is:

  1. Press Save Current Settings so that you can modify settings and then easily return to the global ones.
  2. Change settings in the appropriate dialog, menu item, or control panel.  Menu accelerator keys are Alt-N-M-U to open the Autocontouring Grid Squares dialog, Alt-N-M-N for the Hole Finder dialog, and Alt-N-M-M for the Multiple Record Setup dialog.  When you make changes in those dialogs, they do not need be closed afterwards.
  3. Press Use Current for Grid to save the settings for that grid.
  4. To apply the same settings to other grids, select each other grid in turn in the Multiple Grid Operations dialog and press Use Current for Grid.
  5. Press Restore Current to put the saved settings back in place.

When a grid has settings, the third button is labeled Apply Grid as Current.  Pressing it both saves the current settings and imposes the stored ones, which can be modified.  Thus, to modify stored settings, you go through the same sequence of steps as just described.

General Settings

This category currently captures three settings:

  • Whether to disable trimming of dark borders when doing various alignments, a set of three values set with the Disable Dark Trim in Align
    • item in the Tasks - Specialized Options menu.
  • Whether to erase periodic correlation peaks in general alignments and focusing, set with a checkbox in the Image Alignment & Focus panel.
  • Whether to erase periodic correlation peaks in Realign to Item, set with the Erase Periodic Peaks in LD item in the Navigator menu.
