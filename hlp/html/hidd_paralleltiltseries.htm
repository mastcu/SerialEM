Parallel Tilt Series Setup Dialog

This dialog allows you to set parameters, measure specimen angles, and create target areas for a parallel tilt series.

Magnification for

If not in Low Dose mode, you can set the magnifications to be used for the area map and for acquisition. This section is hidden in Low Dose Mode, in which case Preview will be used for acquisition.

Defining Specimen Angles

The Define Points to Fit Plane routine will measure the defocus at points defined on an area, and then compute a least squares fit with those measurements to determine the geometry of the specimen plane. After points are defined as navigator items, the procedure runs automatically and will delete those items. In other words, all points used for this procedure are temporary and there is no option to continuously update the sample points used for the fit to plane.

  1. Click Define Points to Fit Plane and mark at least four points on an area map. These points should span perpendicular directions on the specimen plane.
  2. Click Stop Adding and Fit. The program realigns to a point in the center of the marked points and switches to the acquisition magnification.
  3. The program image shifts to each point and measures the defocus.
  4. The program computes leveling tilt and X pitch angle, and the corresponding edit boxes in the dialog are updated with these new values.

Define Points to Fit Plane

Click this button to add points spanning the specimen plane. Upon clicking "Stop Adding and Fit", the program will realign to an automatically generated center point, image shift to each drawn point, and measure the defocus at each location. The defocus measurements will then be used to calculate the tilt and X-axis pitch angle of the specimen.

Leveling tilt

The tilt angle at which the specimen will be flat. When starting a new target area, the tilt will be set to this angle.

X pitch angle

The pitch angle of the specimen relative to the X-axis.

Defining Area & Targets

Add Custom Targets / Add Regular Pattern

With Add Custom Targets selected, targets are created by marking specific locations on the area map.

With Add Regular Pattern selected, targets are created according to the pattern set in the multishot parameters, by one of two ways: if the currently selected navigator item is a point, then the targets will follow the multishot pattern centered at the point. If the item is a polygon, the polygon interior will be populated with targets that follow the multishot pattern angle and spacing.

Rough Eucentricity

Runs the Rough Eucentricity routine.

Eucentricity by Focus

Runs the Eucentricity by Focus routine.

Start New Target Area

Click this button to go to the specified leveling tilt and start a new target area. Starting a target area and making an area map are required steps before you can add targets to refine/adjust.

Acquire

In Low Dose mode, there are four acquire buttons: Search, View, Preview, and Montage, which capture images with parameters from the specified Low Dose area. Preview is always used to refine target image shifts and create new adjusting transforms.

If not in Low Dose mode, the four buttons are View, Trial, Preview, and Montage. Clicking View, Trial, or Montage will capture images with the mapping magnification set at the top of this dialog. Equivalently, Preview will use the specified acquisition magnification.

Save/Identify Area Map

If there is a newly captured image or montage in the active buffer, a new map will be created from the active buffer. The image will be saved to the current open file, if possible, or else you will be prompted to open a new file to save the map. Alternatively, if an existing map is loaded into the active buffer, that will be assigned as the area map for the parallel tilt series area.

Adjusting Transform Controls

The parallel tilt series setup dialog provides controls to create and apply adjusting transforms. The routine is as follows:

  1. Click Add Targets and mark points on an area map. These should be recognizable features that you can accurately align by adjusting the image shift at the specified acquisition magnification.
  2. Click Stop Adding & Adjust / Adjust IS. The program realigns to the first marked point and captures an image at the acquisition magnification.
  3. Adjust the image shift to align the original intended target. Click Save Adjustment when done.
  4. The program image shifts to the next point and captures an image at the acquisition magnification.
  5. Repeat steps 3-4 until you have adjusted all image shifts. You can also add more points after saving a round of image shift adjustments, in which case you will be prompted to immediately adjust their corresponding image shifts.
  6. Click Save Transform when you are finished.

The procedure to create a new adjusting transform closely resembles the generic routine to refine target image shifts, which is outlined further below.

Adding Targets, Refining Image Shifts, and Saving Target Maps

The procedure to refine target image shifts can be modified with a variety of options in the dialog, but the generic routine is as follows:

  1. Click Add Targets and mark targets on an area map.
  2. Click Stop Adding & Refine / Refine IS.The program realigns to the target that comes first in the navigator, and captures an image at the acquisition magnification.
  3. Adjust the image shift to align the desired target. Click Save IS / Save Target Maps when done.
  4. The program image shifts to the next point and captures an image at the acquisition magnification.
  5. Repeat steps 3-4 until you have refined all image shifts (and saved a map for each target, if you are using Preview maps for reference). You can also add more targets after refining a round of existing targets, in which case you will be prompted to immediately refine the image shifts of the new targets.
  6. Click Finalize Target Area when you are finished.

Alignment at starting tilt

The option selected here determines which method will be ensure images are aligned to the targets at the starting tilt. Accordingly, the desired option should be selected before refining targets, because they introduce the following modifications to the image shift refinement routine:

  • Take Preview for reference: After image shift is refined, the program saves a map of the Preview image centered on the target.
  • Extract reference from area map: Instead of a Preview map, a crop of the area map centered on the target will be used for alignment.
  • No alignment: The refined image shifts will be applied without any additional alignment.

Note that the latter two options rely on the assumption that the refined image shifts match with target locations marked on the area map. On the other hand, if you are taking Previews for reference, you can modify target locations during the refinement process, since the Preview maps are saved at the same time as the refined image shifts.

Skip image shift refinement

If checked, image shifts will be obtained by converting stage coordinates of the targets drawn on the area map. This means that, if you are saving Preview maps for references, the refinement procedure will run and automatically save all the target maps without stopping and asking you to refine the image shifts. Alternatively, if you are extracting references from the area map or doing no alignment, you can finalize the target area as soon as targets are added, skipping the refinement procedure entirely.

Add Targets / Stop Adding

Click this button to begin adding targets to the area map, or, if already adding, click to stop adding without initiating the refinement routine.

Refine IS / Stop Adding & Refine

Click this button to begin the refinement routine on added points.

Save Target Map/IS

After the routine realigns or image shifts to a target and captures a Preview image, you will be able to freely change the image shift and take more images and this button will be enabled. When you are centered on your desired target, click this button to save the image shift (and possibly also a target map) and continue on to the next target.

Remove Target

After the routine realigns or image shifts to a target and captures a Preview image, you will be able to freely change the image shift and take more images and this button will be enabled. Clicking this button will remove the current target. If it is the first target in the group, the routine will stop so you can rearrange or edit the rest of the added targets. Otherwise, it will continue on to the next one.

Finalize Target Area

Clicking this button will consolidate all refined targets into one parallel tilt series item that contains all necessary information about the image shifts and relevant options from the dialog. The first time a parallel tilt series item is created in a given navigator, you will be prompted to select a tilt series file and the Tilt Series Parameters dialog will open with relevant parameters imported from this dialog.

Abort Area / Abort Refining

Click this button to either clear the whole area and start over, or, if some targets have been refined, just clear the saved image shifts so that refinement can be redone.

Setup Tilt Series

Opens the Tilt Series Parameters dialog for the finalized parallel tilt series item. This is equivalent to selecting the item in the navigator and clicking the TS Params button in the navigator.

Acquisition & Display Options

This closable section of the dialog contains extra options that affect how beam circles are drawn during target area setup, and how which steps are taken during final acquisition.

Assumed max tilt angle for beam display

Beam circles will be drawn as ellipses, stretched perpendicular to the tilt axis based on the tilt angle specified here. Beam circles/ellipses are drawn at all targets while setting up a target area.

Diameter of circles to draw

The diameter of the circles is set to the entered value in microns, unless the option described next is selected to use the illuminated area on a Thermo/FEI Titan or a beam size computed from a beam size calibration on other scopes. Press tab or Enter to have the image redrawn after entering a new diameter.

Use Record illuminated area or beam size for circles in Low Dose

This option now appears on all scopes and allows the multi-shot circle display to take its diameter from the intensity setting of the Record area when Low Dose mode is on instead of from the value entered in this dialog. On a Thermo/FEI Titan, the illuminated area is used for the size and a size will always be available. On other scopes, a beam size calibration is used if one exists for the current conditions (namely, on the same side of crossover and for the same probe mode on Thermo/FEI or alpha on a JEOL). This option is permanently disabled if there are no beam size calibrations at all, or temporarily disabled when there is no calibration for the current Record settings.

For this option to be valuable rather than misleading, the illuminated area readout or calculated beam size must be fairly accurate in the range of interest, which may not be the case.

Multiplier of usual delay after shifting to target

SerialEM has a built-in settling time after image shift that depends on the size of the shift; it is probably inadequate at high magnification. The factor entered here increases this settling time when shifting between targets.

Adjust beam tilt & astig to compenate for image shift

This option is enabled if you have calibrated the beam tilt needed for coma-free alignment as a function of image shift with Coma vs. Image Shift in the Calibration - Focus & Tuning submenu.  If so, then beam tilt will be adjusted appropriately for each Record position, and restored at the end.  As of SerialEM 3.8, the dependence of astigmatism on image shift is also calibrated and will be compensated, unless the property 'SkipAstigAdjustmentForIS' is set. In addition, the image shift present when starting the routine will also be compensated for, unless the script command 'AdjustBeamTiltforIS' was run or beam tilt was adjusted by any of the image-shift setting commands that provide the adjustment.  The conditions under which the calibration was taken are shown on the following line.  For the adjustment to work accurately, the current beam intensity at least should be close to what it was when the calibration was done.

CTF Type

Indicate which CTF type to use, if any, and whether to find astigmatism during final acquisition.

Maximum alignment shift at targets

Enter the maximum shift, as a percentage of the field of view, that will be allowed in alignment to targets.

Aligning to extracted references

If Extract reference from area map is selected for the alignment at starting tilt, then the folllowing additional options will appear:

  • Max rotation: the maximum angle in degrees by which high magnification images at targets will be rotated in order to align to the reference extracted from the area map.
  • Max scaling: the maximum scaling, as a percentage, to be applied to high magnification images at targets in order to align to the reference extracted from the area map.
