Parallel Tilt Series Setup Dialog

This dialog allows you to set parameters, measure specimen angles, and create targets for a parallel tilt series.

Magnification for

If not in Low Dose mode, you can set the magnifications to be used for the area map and for acquisition. This section is hidden in Low Dose Mode, in which case Preview will be used for acquisition.

Defining Specimen Angles

Define Points to Fit Plane

Click this button to add points spanning the specimen. Upon clicking "Stop Adding and Fit", the program will realign to an automatically generated center point, image shift to each drawn point, and measure the defocus at each location. The defocus measurements will then be used to calculate the tilt and X-axis pitch angle of the specimen.

Leveling tilt

The tilt angle at which the specimen will be flat. When starting a new target area, the tilt will be set to this angle.

X pitch angle

The pitch angle of the specimen relative to the X-axis.

Defining Area & Targets

Add Custom Targets / Add Regular Pattern

With "Add Custom Targets" selected, targets are created by marking specific locations on the area map.

With "Add Regular Pattern" selected, targets are created according to the pattern set in the multishot parameters, by one of two ways: if the currently selected navigator item is a point, then the targets will follow the multishot pattern centered at the point. If the item is a polygon, the polygon interior will be populated with targets that follow the multishot pattern angle and spacing.

Acquire

In Low Dose mode, there are four acquire buttons: Search, View, Preview, and Montage, which capture images with parameters from the specified Low Dose area. Preview is always used to refine target image shifts and create new adjusting transforms.

If not in Low Dose mode, the four buttons are View, Trial, Preview, and Montage. Clicking View, Trial, or Montage will capture images with the mapping magnification set at the top of this dialog. Equivalently, Preview will use the specified acquisition magnification.
