Title Bar

The title bar is located along the top of a window. It contains the name of the application and open image file.

To move the window, drag the title bar. Note: You can also move dialog boxes by dragging their title bars.

A title bar may contain the following elements:

  • Application Control-menu button
  • Image window Control-menu button
  • Maximize button
  • Minimize button
  • Name of the application
  • Name of an open image file
  • Restore button
