File Save As dialog box

The following options allow you to specify the name and location of the file you're about to save:

File Name

Type a new filename to save a document with a different name. SerialEM adds the extension indicated in the Save File As Type box unless you enter some other extension recognized by the system.  In the case of an MRC file, it will add the extension '.mrc', but not if you enter '.st' or '.map'.

Drives

Select the drive in which you want to store the document.

Directories

Select the directory in which you want to store the document.

Network...

Choose this button to connect to a network location, assigning it a new drive letter.
